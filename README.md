## Задача 2

### Текст задачи

Возвратить true, если в числе есть цифра k, и false — в противном случае

### Алгоритм решения

Для решения используется рекурсивная функция containsDigit
1) В функцию передаются исходное число number и искомая цифра k
2) Если число равно 0,то это значит,что мы перебрали все цифры(или оно таким и было). Возвращаем true, если k = 0. false в других случаях
3) Рекурсия:
    Извлекаем последнюю цифру числа с помощью операции взятия остатка от деления на 10: abs(number % 10)
    Если последняя цифра равна искомой k, возвращаем true
    Иначе запускаем функцию заново для числа, отбрасывая последнюю цифру (number / 10)
4) В основне запрашиваются число и цифра, выводится результат проверки

## Задача 3

### Текст задачи

Создайте собственные функции для выполнения основных операций над списками (добавление/удаление/поиск элемента, сцепка двух списков, получение элемента по номеру).

### Алгоритм решения

Для решения были созданы функции:
1)  getItem index list: Получение элемента по индексу
       Если список пуст([]), то ошибка индекса вне диапазона
       Если индекс равен 0, то возвращаем голову списка (head)
       Иначе рекурсивно вызываем функцию для хвоста (tail) с индексом index - 1
2)  addToEnd element list: Добавление элемента в конец
       Если список пуст, то возвращаем новый список из одного элемента ([element])
       Иначе возвращаем список, где голова это текущая голова, а хвост это результат рекурсивного добавления элемента в хвост первого списка
3)  removeByValue value list: Удаление первого вхождения элемента по значению
       Если список пуст,то возвращаем пустой список
       Если голова списка равна искомому значению, то возвращаем хвост списка удаляя голову
       Иначе возвращаем список, где голова это текущая голова, а хвост это результат рекурсивного удаления элемента из хвоста
4)  contains element list: Проверка наличия элемента в списке
       Если список пуст,то элемент не найден (false)
       Если голова равна искомому элементу , то элемент найден (true)
       Иначе рекурсивно ищем элемент в хвосте списка
5)  append list1 list2: Сцепка двух списков (конкатенация)
       Если первый список пуст,то результатом является второй список
       Иначе возвращаем список, где голова это голова первого списка, а хвост это результат рекурсивной сцепки хвоста первого списка со вторым списком
6)  length list: Получение длины списка
       Если список пуст, то возвращаем 0
       Иначе возвращаем 1 + длина хвоста

Для взаимодействия с пользователем создано консольное меню, где можно выбрать операцию и ввести данные
